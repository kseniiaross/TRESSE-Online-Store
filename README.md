# TRESSE — Full-Stack E-Commerce Platform

TRESSE is a modern full-stack e-commerce application built with a focus on **accessibility (WCAG 2.1)**, **performance**, and **scalability**.  
Stack: **React (TypeScript)** + **Redux Toolkit** + **Vite** on the frontend and **Django REST Framework** + **PostgreSQL** on the backend.

## Key Features
- Product catalog with filters, search, and detail pages  
- Wishlist (favorites)  
- Shopping cart and checkout (guest and authenticated users)  
- Product reviews and ratings (CRUD)  
- JWT authentication (register/login/logout)  
- Responsive design + accessibility (WCAG 2.1)  
- Fast builds and great DX with Vite + TypeScript  
- Clean API architecture with DRF and PostgreSQL as the main database  
- Ready for CI/CD and testing (Jest/Pytest on roadmap)

## Tech Stack
**Frontend**
- React 18, TypeScript, Redux Toolkit  
- React Router, Axios    

**Backend**
- Python 3, Django, Django REST Framework

**DevOps & Tooling**
- GitHub Actions (CI/CD)  
- Netlify/Vercel (frontend)

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
